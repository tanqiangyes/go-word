## 🔍 快速查找

### 常用功能
- **打开文档**: `word.Open()`
- **获取文本**: `doc.GetText()`
- **获取段落**: `doc.GetParagraphs()`
- **获取表格**: `doc.GetTables()`
- **创建文档**: `writer.NewDocumentWriter()`
- **保存文档**: `docWriter.Save()`

### 高级功能
- **表格操作**: `NewAdvancedFormatter()`
- **文档保护**: `NewDocumentProtector()`
- **批量处理**: 查看示例代码
